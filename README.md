# employee-attendance-management-system
Employee Attendance management System built using Core Java , Spring Boot , MySQL , Hibernate , and Spring Data JPA.


##  Project Description
This project is developed using Spring Boot and MySQL to manage employee attendance records.

##  Technologies Used
- Core Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- REST APIs

##  Features
- Add Employee
- Mark Attendance
- View Attendance
- Update Employee
- Delete Employee

## ▶️ How to Run
1. Clone the repository
2. Configure MySQL in application.properties
3. Run the Spring Boot application
